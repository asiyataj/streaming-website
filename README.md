# streaming-website
A full-stack web application for browsing, searching, and watching movies online. Users can create accounts, stream videos, and manage watchlists — just like a mini Netflix clone.

📌 Key Highlights
✅ User Authentication (JWT)

🔎 Movie Search & Filter

📺 Video Streaming Integration

❤️ Watchlist / Favorites

🧠 Admin Panel (Add/Edit Movies)

📱 Fully Responsive Design

💡 Purpose
This project was built to:

Showcase a full-stack application using React, Node.js, and MongoDB

Demonstrate user authentication and role-based access (admin/user)

Implement cloud video storage and playback functionality

Provide a real-world example of building a dynamic, media-driven UI

Explore RESTful API design and media streaming techniques

🔐 Features At a Glance
Create and manage user accounts

Search, browse, and filter movies

Stream videos in-browser

Save movies to personal watchlist

Admin dashboard to upload/edit/delete content

Mobile-first responsive layout
